
# SpecAi

Front end ui development of a introduction website

## Technology Stack

- React JS
- TypeScript
- Tailwind CSS
- Docker

### Libraries Used

 - React Fast Marquee
 - Swiper
 - Framer motion


## Run

Run Docker Desktop

Run below two commands in a terminal

- docker build -t specai:dev .


- docker run -p 5173:5173 specai:dev
